#### Variables

* There must be no spaces on either side of the equals sign
* whitespace characters: a space, a tab, a newline character，must quote(引号) the character
   
#### Control Structures

* if ... then ... else ... fi

```
if [ -f test.c ]    # [] equivalent test => if test -f test.c 
then
    echo "Is file."
else 
    echo "Is not file."
fi
```
